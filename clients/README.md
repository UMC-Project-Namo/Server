## Global Rule

- `Clients` 모듈은 프로젝트 외부 서비스의 API를 호출하는 기능을 제공한다.
- `Clients` 모듈은 서비스의 Business/Domian Logic을 알 수 없다.

## Social-Client Module

- `:clients:social-client` 모듈은 각종 소셜 로그인과 관련된 외부 서비스의 API를 호출하는 기능을 제공한다.

### 제공되는 기능

- Social Login Client
  - Apple
  - Kakao
  - Naver
